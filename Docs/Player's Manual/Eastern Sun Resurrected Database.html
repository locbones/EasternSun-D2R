Eastern Sun Resurrected

DIFFERENCES WITH CLASSIC EASTERN SUN

- Added a third cavern other than Gfrazier's Dome and Bill Roper's Memorial Ball Park
- Added a few new magic affixes
- Anya's reward (Anya's gift in in classic ES) is now dropped by Act5 Ancients and it's called "Ancient's gift" (That's an hardcoded feature that can't be modified in D2R)
- Better loot in Naraku and Nihlathak's Domain
- Disabled normal gems to drop after Nightmare difficulty
- Dragonstones cannot be transmuted with a click right mouse's button (but you can transmute them simply by cubing them, up to 3 at once)
- Extra skills from quests are now in form of dropped elixirs from Blood Raven (+1), Radament (+1) and Diablo (+2). (That's an hardcoded feature that can't be modified in D2R)
- Inventory is 10x8 instead of 10x9
- Kanji runes needs only 1 crystal instead of 2 (except God and Heaven runes)
- Nihlathak's Domain is now 2 levels with first level being large and the second smaller with all the superunique bosses (replicate the domain's like in original ES is an impossible task, sorry)
- No improved reward from Hellforge (That's an hardcoded feature that can't be modified in D2R)
- No Qual Khek's reward after rescuing barbarian prisoners (That's an hardcoded feature that can't be modified in D2R)
- Removed ancient scrolls: recipes that required an ancient scroll now requires an ancient decipherer in its place
- Removed gambling chit being now obsolete thanks to the D2R gambling system
- Removed killcounter
- Removed magic eraser (it causes some troubles, may return in next updates if fixed)
- Removed tokenizer (it causes crashes in D2 Resurrected)
- Server only runewords enabled
- Some monster names changed if they don't match their new models
- Some skills improved
- Some new horadric cube recipes
- Some unique items (both ES and LoD) improved
- Splash damage added to Damage Augmenter
- Uber Quest enabled
